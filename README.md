# Launch

Build digital products and get paying customers. Three skills, one loop.

```
/ship      → build the product fast
/launch    → blast it across all channels on day one
/pipeline  → weekly engine that keeps clients coming in
```

## The Loop

**Build** — `/ship` turns a spec into working code in days.

**Launch** — `/launch` fires 5 content agents in parallel on day one:
Twitter thread, LinkedIn post, Reddit post, Product Hunt copy, SEO audit.
Post-bridge schedules distribution. PostHog tracks what converts.

**Acquire** — `/pipeline` runs every week:
generates content from recent work, finds 10 leads, writes personalized emails,
sends follow-ups. Keeps the client pipeline moving between launches.
